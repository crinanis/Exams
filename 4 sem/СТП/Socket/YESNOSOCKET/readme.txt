Многопоточное клиент-серверное приложение, то есть сокеты, несколько клиентов.
 Клиент может посылать сообщения серверу с текстом "/yes", "/no", "/getall", "/help". 
Остальные сообщения сервером должны игнорироваться. По команде "/getall" сервер должен выводить количество yes и no, 
которые были отправлены клиентом до этого момента, по команде "/help" выводиться список доступных комманд
