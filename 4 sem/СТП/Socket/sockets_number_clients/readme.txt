СО СЛОВ СДАЮЩЕГО

Создать клиент серверное многопоточное приложение на сокетах.
Клиенты посылают серверу любые сообщения, после этого сервер посылает каждому клиенту целое число,
соответствующее порядковому номеру подключившегося клиента.
Первому на все сообщения посылается 1 второму - 2.
Продемонстрируйте работу для трех клиентов и одного сервера.
